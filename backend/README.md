# SL Cosmetica Backend

Production-ready Node.js + Express API with PostgreSQL for SL Cosmetica e-commerce.

## Features
- JWT authentication with bcrypt password hashing
- PostgreSQL database with proper indexes
- Rate limiting and security headers
- CORS configuration
- User management with addresses and preferences

## Environment Variables

```env
NODE_ENV=production
PORT=10000

DB_PORT=5432

DB_USER=root
DB_PASSWORD=<your-password>

JWT_EXPIRES_IN=7d
FRONTEND_URL=<your-frontend-url>
```

## Deployment

### Render
1. Push to GitHub
2. Connect repository to Render
3. Set environment variables
4. Deploy
5. Run migration: `npm run migrate`

## API Endpoints

### Public
- `POST /api/auth/register` - Create account
- `POST /api/auth/login` - Sign in

### Protected (requires Bearer token)
- `GET /api/auth/me` - Get current user
- `PUT /api/auth/profile` - Update profile
- `POST /api/auth/addresses` - Add address
- `PUT /api/auth/addresses/:id/default` - Set default
- `DELETE /api/auth/addresses/:id` - Delete address

## Local Development

```bash
npm install
npm run migrate  # One time - create tables
npm run dev      # Start server
```
